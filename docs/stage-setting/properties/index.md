# Properties

{term}`Properties <Property>` are used to describe the characteristics of objects, or {term}`prims <Prim>` within an OpenUSD scene. Prims can have two types of properties: {term}`attributes <Attribute>` and {term}`relationships <Relationship>`. Attributes store data values that define the appearance, behavior, or other characteristics of a prim. Relationships, on the other hand, establish connections between prims or properties, enabling hierarchical structures and data sharing.
